TROUBLE SHOOTING

When I logon, I get a loading spinner on my screen for along time

  • If you are logging in for the first time, or a version release has been applied to the Application, the browser can remember old information which prevents the user form logging on successfully. Press the browser refresh icon and the problem should be resolved.
  • Else, if that doesn’t work, clear your browser’s Cache and try again.

I have reset my password in Maximo, now I can’t logon to the VI Portal

  • Goto the your user account settings, select 3rd party details, and update the password details.

I have reset my password in the 3D Viewer, now I can’t logon to the VI Portal

  • Goto the your user account settings, select 3rd party details, and update the password details.

I can’t filter my asset by the location in the 3D view

  • Is the Location Hierarchy in a Building->Floor->Room structure?
  • If the Maximo Location Hierarchy is built around the asset hierarchy, then the level that relates to the room in which each asset is housed must be mapped to the Room in the VI Site Model.

The Asset Register takes a long time to load

  • How Many Assets are held in the Asset Register?
  • By mapping the location hierarchy to the site model, the asset register filters the assets by the location the user is currently located within the 3D View., Else the entire Asset Register will be requested form Maximo. If the Assets Register is large, this can use significant bandwidth and take some time to process.

All my assets are classified as Unknow in the 3D View

  • How are the Assets classified in Maximo?
  • using the Asset Types domain or via user definable attribute in the Maximo database?
  • In the installation manifest, amend the asset types object to be the name of the attribute that stores the asset type.

I cant see my IoT data in Maximo Meters

  • If Asset Meters are deployed, ensure the deviceId is stored in the DEVICEID attribute. This is the device that broadcasts the data points (asset meters). For more assistance, contact VI Support.

I cant access a feature of the VI App when I right click

  • Check the memberships in the Application Manager. You will need ADMIN or SUPER rights to do this. Ensure the Module you are looking for is turned on against the users account.

I cant access the Infohub

  • Check the memberships in the Application Manager. You will need ADMIN or SUPER rights to do this. Ensure the Infohub Module is turned on against the users account.
